## How to install

* Download or Clone this repo
* Run this command in the root and frontend floder
```
npm install
```
* Create `.env` file in root
* `.env` example
```
MONGO_URI=
NODE_ENV=
PORT=
JWT_SECRET=
```
* Import seed data
```
npm run data:import

```
* Run this command
```
npm run dev
```
